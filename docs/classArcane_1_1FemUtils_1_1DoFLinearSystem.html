Detailed Description

Linear system Matrix A + Vector x + Vector b for Ax=b.

Before using an instance of this class you need to call method initialize(). If you want to reuse the same instance for several solving you need to call reset() to destroy the underlying linear system and then you need to call initialize() again.

The solve() method solves the current linear system. After this variable returned by the method solutionVariable() will be filled with the values of the solution vector.
