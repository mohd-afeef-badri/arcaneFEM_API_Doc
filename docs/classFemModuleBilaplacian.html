void FemModuleBilaplacian::_assembleBilinearOperator ( )
private

Assembles the bilinear operator for the FEM simulation.

This method constructs the bilinear operator by iterating over all cells and computing the element matrix for each cell. The values are then added to the global matrix.

Note: The method assumes that the linear system has been initialized before calling this function.

◆ _assembleLinearOperator()

void FemModuleBilaplacian::_assembleLinearOperator ( )
private

Assembles the linear operator for the FEM simulation.

This method constructs the linear system by assembling the LHS matrix and RHS vector, applying various boundary conditions and source terms.

Steps involved:

  1. The RHS vector is initialized to zero before applying any conditions.
  2. If Neumann BC are specified applied to the RHS.
  3. If Dirichlet BC/Point are specified apply to the LHS & RHS.

Computes the element matrix for a triangular element (ℙ1 FE).

This function calculates the integral of the expression: a(𝐮,𝐯) = ∫∫ (∂𝑢𝑥/∂𝑥 ∂𝑣𝑦/∂𝑥 + ∂𝑢𝑦/∂𝑥 ∂𝑣𝑦/∂𝑦)dΩ

  • ∫∫ (𝑢𝑦 𝑣𝑦)dΩ

  1. Calculate the area of the triangle.
  2. Compute the gradients of the shape functions.

void FemModuleBilaplacian::_doStationarySolve ( )
private

Performs a stationary solve for the FEM system.

This method follows a sequence of steps to solve FEM system:

  1. _assembleBilinearOperator() Assembles the FEM matrix 𝑨
  2. _assembleLinearOperator() Assembles the FEM RHS vector 𝒃
  3. _solve() Solves for solution vector 𝒖 = 𝑨⁻¹𝒃
  4. _updateVariables() Updates FEM variables 𝒖 = 𝒙
  5. _validateResults() Regression test

◆ compute()

void FemModuleBilaplacian::compute ( )
override

Method called at the beginning of the simulation.

Performs the main computation for the FemModuleBilaplacian.

This method:

  1. Stops the time loop after 1 iteration since the equation is steady state.
  2. Resets, configures, and initializes the linear system.
  3. Applies PETSc commandline flags to the solver (if PETSc is used).
  4. Executes the stationary solve and extracts psi.
